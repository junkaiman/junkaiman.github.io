用acme.sh申请SSL证书

Feb 18 2020

背景

之前用acme.sh只能申请一级域名的SSL证书，二级域名就申请不下来，各种提示错误…最近要搭网盘，要搭Jupyter，就赶紧学学怎么把二级域名SSL证书都申请下来。

操作

根据acme的dnsapi官方文档，

export VULTR_API_KEY="<Your API key>"

curl https://get.acme.sh | sh
        ~/.acme.sh/acme.sh --issue --dns dns_vultr -d thekeon.tk -d '*.thekeon.tk'
            ~/.acme.sh/acme.sh  --installcert  -d  codingkeon.com   \
            --key-file   /usr/src/all-cert/private.key \
            --fullchain-file /usr/src/all-cert/fullchain.cer \
            --reloadcmd  "systemctl force-reload  nginx.service"

验证

看到secure了!
